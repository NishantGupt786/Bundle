Syntax for all the functions and methods you need

Print statement
print(#statement)

Note that strings are embedded in "" and integers and floats can be printed plain.

We can use formatted strings to make printing much easier for which the syntax is:

print(f"My name is {name}")

Conditional Statements

Format of the if.... elif..... else Statements is as follows:

We can also use nestes if Statements where there is an if statement inside another if statement.

if(name == "Nishant"):

      print("Correct")

elif(name == "david"):

      print("Incorrect")

else:

      print("Please enter valid name")

Loop Statements

In Python there are two types of loop Statements: the "while" and the "for" loop

Here is the syntax of the while loop

i = 1

while i < 6:

      print(i)

      i += 1

Here is the syntax of the for loop

for i in range(5):

      print(i)

      

We can also use the for loop to iterate over strings and lists.

We can also nest loops where one for loop is inside another or one for loop is inside a while loop.

Defining functions

In python we can define functions which we can later call in our program to do some particular task.

The syntax for the following is:

def square(x):

      return (x*x)

num = square(5)

print(num)

This particular function takes in one integer "x" and returns its squared value.

Note that to call this function we just type the name of the function followed by the parameter in paranthesis.

Importing libraries and important python libraries I use

A library is code that another person has written which you can use in your program to use the functions of his code.

There are many ways to import a function from a library which you can see as follows

from math import sqrt

      print(sqrt(25))

import math

      print(math.sqrt(25))

from math import sqrt as root

      print(root(25))

Now the libraries that I use regularly are the numpy, selenium and math libraries.

The selenium library is used to make a web automation program that does some function for you in the web by only running a program. I have used this to order food online through swiggy without actuall doing anything.

The numpy library is used to handle large stes of data which can be stored in 2-d arrays which becomes very complicated and hard if done normally on python.

The math module contains various mathematic and logical operations which I use at a daily basis.
