Set up

Now, to use jupyter we are going to download a platform called anaconda which contains all the libraries and modules required to start off with machine learning.

To download anaconda head over to this link:

anaconda individual installer

After downloading and following the on-screen procedures head to your start button and seach for anaconda navigator.

Open it and launch Jupyter notebook

In the desktop directory create a new python file through your notebook and you are good to go.

The Gist of Machine Learning

Machine learning comprises of 6 steps:

importing the data, splitting the data into test/train sets, creating a model, training the model, making predictions, evaluating and improving

Step 1: Importing the data

To import the data we can do the following steps:

import panda as pd

variablename = pd.read_csv('csvfile.csv')

Step 2: Splitting the data into test/train sets

To split data you can use the following steps:

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

What this does is that it splits 20% of the data for testing and 80% for training

Step 3: Creating a model

To create a model follow these steps:

X = variablename.drop(columns=['output'])

y = variablename['input']

Step 4: Tain the model

To train the model we can:

from sklearn.tree import DecisionTreeClassifier

= DecisionTreeClassifier()

model.fit(X_train, y_train)

Step 5: Making predictions

To make predictions we need to run the following:

predictions = model.predict(X_test)
Step 6: Evaluating and improving

To evaluate the predictions we can run the following Statements:

score = accuracy_score(y_test, predictions)

print(score)

To improve the model we can tweak the distribution of the train/test sets until we get favourable responses.
